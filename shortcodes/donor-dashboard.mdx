The donor dashboard shortcode renders a modern, tab-based dashboard for logged-in donors. It includes their giving stats, full donation history, recurring subscriptions, annual receipts (when the module is enabled), and a profile editor.

## Basic usage

Drop the shortcode on any page:

```text
[donorpress_dashboard]
```

## Customizing the accent color

The dashboard uses a single accent color for active tabs, buttons, badges, and form focus states. Pass any color via the `accent_color` attribute:

```text
[donorpress_dashboard accent_color="#99BADD"]
```

The attribute accepts **named CSS colors, HEX codes, or RGB codes**. All three of the following produce a true red accent:

```text
[donorpress_dashboard accent_color="red"]
[donorpress_dashboard accent_color="rgb(255,0,0)"]
[donorpress_dashboard accent_color="#ff0000"]
```

<Tip>
	The dashboard auto-picks a readable button text color (black or white) based on the luminance of your accent. So a pale yellow accent gets dark text, a deep navy gets white — no manual contrast tuning needed.
</Tip>

### Behind the scenes

The accent color is injected as a CSS variable (`--dp-dd-accent`) on the dashboard's root element. Every interactive surface — tabs, buttons, badge tints, focus rings — derives from that single value, so theming is consistent.

## All shortcode attributes

| Attribute | Type | Default | Description |
| --- | --- | --- | --- |
| `accent_color` | string | `#3858e9` | Accent color used for tabs, buttons, and focus states. Accepts HEX, RGB, or named CSS color. |

## Login & no-data states

The shortcode handles two empty states out of the box:

- **Not logged in** — shows a centred "Please log in to view your donor dashboard" message with a link to the WordPress login screen, redirecting back to the dashboard page after sign-in.
- **No matching donor record** — shows a "No donation history found" message. This appears for logged-in users whose email doesn't match any donor in your database.

Both states reuse the same accent color you configured on the shortcode.

## URL hash navigation

The dashboard syncs the active tab to the URL hash (`#dp-overview`, `#dp-donations`, `#dp-subscriptions`, `#dp-annual-receipts`, `#dp-profile`). This lets you:

- Link directly to a specific tab from emails or other pages, e.g. `https://yoursite.com/dashboard/#dp-subscriptions`.
- Bookmark a particular view.
- Share a link that opens on the right tab.

## Where to place it

We recommend creating a dedicated **"My Account"** or **"My Donations"** page and adding `[donorpress_dashboard]` as the only content. Then link to it from:

- Your site's main navigation (visible only to logged-in users).
- The donation receipt email (`[receipt_link]` template tag).
- The thank-you page after a successful donation.
