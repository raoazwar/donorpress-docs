---
title: "Filters"
description: "Reference for every WordPress filter DonorPress exposes for runtime customization of donation data, forms, emails, gateways, and templates."
---

Filters let you modify DonorPress data and behaviour without forking the plugin. All filters use the `donorpress/` prefix.

## Form filters

| Filter | Modifies | Args |
| --- | --- | --- |
| `donorpress/form/field_types` | Available field types in the form builder. | `array $types` |
| `donorpress/form/before_render` | The form data before rendering. | `array $form_data` |
| `donorpress/form/after_render` | The form HTML after rendering. | `string $html, array $form_data` |
| `donorpress/form/submission_validate` | Per-submission validation result. Return WP_Error to reject. | `true\|WP_Error $valid, array $submission` |
| `donorpress/form/spam_check` | Whether a submission is spam. Return true to reject. | `bool $is_spam, array $submission` |

## Donation filters

| Filter | Modifies | Args |
| --- | --- | --- |
| `donorpress/donation/data_before_create` | Donation data array before insertion. | `array $data` |
| `donorpress/donation/refundable` | Whether a donation is refundable. | `bool $can_refund, Donation $donation` |
| `donorpress/donation/receipt_url` | URL for the donation receipt. | `string $url, Donation $donation` |

| Filter | Modifies | Args |

| `donorpress/donor/dashboard_url` | URL for the donor dashboard. | `string $url, Donor $donor` |
| `donorpress/donor/can_view_dashboard` | Whether the current user can view the donor dashboard. | `bool $can_view, Donor $donor` |

## Gateway filters

| Filter | Modifies | Args |
| --- | --- | --- |
| `donorpress/gateways` | The full registry of gateways. Add or remove gateways. | `array $gateways` |

## Examples

### Force all donation receipts BCC'd to your accountant

```php
add_filter( 'donorpress/email/headers', function ( $headers, $template ) {
	if ( $template === 'donation_completed' ) {
		$headers[] = 'Bcc: accountant@yourcharity.org';
	}
	return $headers;
}, 10, 2 );
```

### Match donors by email AND phone (instead of email only)

```php
add_filter( 'donorpress/donor/match_data', function ( $data ) {
	// Add phone to the matching key. Note: requires customising the donor lookup query too.
	$data['_match_by_phone'] = true;
	return $data;
} );
```

### Hide one of the built-in gateways

```php
add_filter( 'donorpress/gateways', function ( $gateways ) {
	unset( $gateways['paypal'] );
	return $gateways;
} );
```

### Custom report cache duration

```php
add_filter( 'donorpress/reports/cache_minutes', fn() => 30 );
```
