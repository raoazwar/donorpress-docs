## Built for WordPress

- **Modern stack** — PHP 8.1+, React 18, Vite, Tailwind, REST API.
- **Lightweight** — no heavy admin shells. The settings UI is a fast SPA, the donor frontend is plain CSS/JS.
- **Extensible** — service-provider architecture, REST endpoints, and a clean hook surface.
- **Translatable** — full text-domain coverage; place translations in `/languages`.

## Free vs. premium

DonorPress is free at its core. Premium add-ons unlock advanced workflows:

| Module | Plan | What it does |
| --- | --- | --- |
| PDF Invoices | Basic | Auto-generate branded PDF receipts. |
| Manual Donations | Basic | Record offline donations (cash, check, bank). |
| Annual Receipts | Basic | Year-end tax receipts with downloadable PDFs. |
| Advanced Dashboard | Premium | KPIs, charts, MRR projections, donor insights. |
| Gift Aid (UK) | Premium | HMRC-compatible Gift Aid declarations and exports. |
| Currency Switcher | Premium | Multi-currency forms with live exchange rates. |

[See all modules →](/modules/overview)
